# Contributing to Calendeo Embed

Thank you for your interest in contributing to Calendeo Embed!

## Reporting Issues

- Use GitHub Issues to report bugs
- Include details: WordPress version, PHP version, plugin version
- Provide steps to reproduce

## Code Standards

- Follow WordPress Coding Standards (WPCS)
- Use 4 spaces for indentation
- Use meaningful variable and function names
- Add proper escaping for output (esc_html, esc_attr, esc_url)
- Sanitize input (sanitize_text_field, etc.)

## Translation

Translations are welcome! We use PO/POT files. To contribute:

1. Download calendeo-embed.pot
2. Create translated .po file (e.g., calendeo-embed-de_DE.po)
3. Submit via GitHub or contact us

If you discover security vulnerabilities, please email kontakt@calendeo.pl instead of using the issue tracker.
